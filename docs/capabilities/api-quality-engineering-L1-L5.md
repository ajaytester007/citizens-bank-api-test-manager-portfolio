# API Quality Engineering Capability Model – L1 to L5

| Level | Capability | Description |
|---|---|---|
| L1 | API Smoke Validation | Basic endpoint availability, response code, authentication, and payload checks |
| L2 | Functional API Validation | REST/SOAP business scenario validation, schema verification, and error handling |
| L3 | Integration & Workflow Testing | API chaining, service orchestration, middleware transformations, and downstream validation |
| L4 | Banking Controls Validation | Payment lifecycle, settlement, reconciliation, financial controls, audit evidence |
| L5 | Enterprise API Governance | CI/CD quality gates, executive dashboards, release certification, Go/No-Go support |

## Key Tools
ReadyAPI, SOAPUI Pro, Postman, REST Assured, SQL, Jenkins, GitHub Actions, Allure, Jira/Xray.
