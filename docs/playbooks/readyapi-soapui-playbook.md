# ReadyAPI / SOAPUI Playbook

## Scope
Use ReadyAPI and SOAPUI Pro for SOAP and REST service validation, WSDL/XSD verification, property transfers, Groovy scripting, mock services, service virtualization, and regression execution.

## Test Design Pattern
1. Import WSDL/OpenAPI specification
2. Define service contract expectations
3. Create functional test cases
4. Add assertions for status, schema, headers, payload, and business rules
5. Configure property transfers and data-driven scenarios
6. Add negative and boundary tests
7. Execute regression suite
8. Export evidence for release certification

## Banking API Examples
- Payment initiation
- Payment status inquiry
- Account balance inquiry
- Settlement confirmation
- Exception handling
- Loan servicing transaction update
